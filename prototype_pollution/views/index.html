🔍 原型污染狀態檢查

🎯 方法執行攻擊

透過 variable[method] 的方式執行動態方法，攻擊者可以輸入 constructor 來污染原型。

⚙️ 設定合併攻擊

透過合併使用者設定時的不安全操作，攻擊者可以使用 __proto__ 來污染原型。

💀 攻擊範例說明

1. Constructor 原型污染

通過 constructor.prototype 污染所有物件：

方法名稱: constructor 參數: [{"prototype": {"isAdmin": true}}]

2. __proto__ 原型污染

通過 __proto__ 直接修改原型鏈：

{ "__proto__": { "isAdmin": true, "role": "admin" } }

3. 深層巢狀污染

通過深層巢狀結構進行污染：

{ "user": { "__proto__": { "polluted": "yes" } } }
